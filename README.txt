움공바이블:새번역 Vercel 배포용 파일입니다.

실제 배포 주소:
https://movech-bible.vercel.app

이 패키지는 위 주소 기준으로 SEO, Open Graph, Twitter Card 미리보기 이미지 주소가 설정되어 있습니다.

사용 방법:
1. 이 ZIP을 압축 해제합니다.
2. 안의 파일들을 GitHub 저장소에 다시 업로드합니다.
3. Vercel이 자동 재배포되거나, Vercel 프로젝트에서 Redeploy를 실행합니다.
4. 배포 후 https://movech-bible.vercel.app 로 접속해 확인합니다.
